# SAS-Curiosity EtuInvestors
For sharing SAS ODA codes We've written for SAS Curiosity Cup 2026.  
Project name: Measuring the Impact of FED Funds, M2 Supply, and DXY on Digital Asset Pricing  
Team -> EtuInvestors  
Authors: Ural Şenyılmaz, Melih Gümüş, Anıl Serdar Ünal  

File Structure:  
SAS-Curiosity/  
├── CODE-ANALYSIS-GRAPHS/    # SAS ODA scripts for analysis and visualization (.sas)  
│   ├── correlation_tables.sas  # code for generating tables of the correlation analysis  
│   ├──CryptoCoins #the code for plotting the cryptocurrencies' price history with macroeconomic parameters  
│   └── Histograms.sas   # Code for generating the histograms  
├── DATASETS/                # Raw and processed data files used in the project  
│   ├── CRYPTO PRICE HISTORY    # Historical cryptocurrency market data  
│   ├── MACROECONOMIC DATA # Macroeconomic parameter datasets (M2 Money Supply, US Dollar Index, FED Funds)  
│   └──ALL_CORRELATIONS_RESULTS.csv #The correlations of the cryptocurrencies and the macroeconomic parameters  
├── README.md                # Project documentation and team information  
└── .gitignore               # Configuration to ignore temporary or local files
